# Python Socket Shell (Educational Demo)

## Purpose
This project demonstrates how low-level socket communication and process control work in Python. It shows how a client can establish a TCP connection and interact with a system shell.

This repository is intended **strictly for educational purposes**, such as:
- Learning about networking with Python sockets
- Understanding how subprocesses and I/O redirection work
- Practicing in controlled lab environments or CTF-style exercises

## Features
- TCP socket connection
- Redirection of standard input/output over network
- Cross-platform shell handling (Windows/Linux)
